Association Between Variables#

We will use the Titanic dataset again as an example:

import pandas as pd
titanic_data = pd.read_csv('https://raw.githubusercontent.com/agconti/kaggle-titanic/master/data/train.csv',

Given the dataset above, we might ask ourselves if some variables influence the values of others. For example, being in first, second, or third class (variable Pclass) somehow influences the probability of surviving (variable Survived)?, or perhaps, does age (variable Age) or the price paid (Fare) somehow influence the probability of survival (Survived)?

In this lesson, we will look at different ways to summarize the distributions of two variables and check for any associations (or correlations) between the variables.

Summarize the Distribution of Two Discrete Variables#

If both variables we are studying are discrete, we can enumerate all possible combinations of values and summarize them in a contingency table that shows the values of one variable on the rows and those of the other variable on the columns. Each cell will indicate the number of times we observe a given pair of values.

Contingency Tables#

Let’s build the contingency table for the variables Sex and Pclass. To do this, we will use the crosstab function of Pandas:

pd.crosstab(titanic['Sex'], titanic['Pclass'])
Pclass 1 2 3
Sex
female 94 76 144
male 122 108 347

The table above indicates, for example, that \(94\) passengers in class \(1\) were female. By specifying margins=True, we can obtain row and column sums, which will indicate the absolute frequencies of each variable:

pd.crosstab(titanic['Sex'], titanic['Pclass'],margins=True)
Pclass 1 2 3 All
Sex
female 94 76 144 314
male 122 108 347 577
All 216 184 491 891

Joint Relative Frequency Distributions#

We can obtain the joint relative frequency table by passing normalize=True to the crosstab function:

pd.crosstab(titanic['Sex'], titanic['Pclass'], normalize=True)
Pclass 1 2 3
Sex
female 0.105499 0.085297 0.161616
male 0.136925 0.121212 0.389450

We obtain the marginals by passing margins=True:

titanic['Sex'].value_counts()

titanic['Sex'].value_counts(normalize=True)

Conditional Relative Frequency Distributions#

It is possible to obtain the conditional relative frequencies by normalizing by the appropriate row. If I want to condition with respect to variable X, I will normalize by rows:

# normalize=0 indicates to condition on the first variable
pd.crosstab(titanic['Sex'], titanic['Pclass'], normalize=0)
Pclass 1 2 3
Sex
female 0.299363 0.242038 0.458599
male 0.211438 0.187175 0.601386

From the table above, for example, I can infer:

  • \(f(Pclass=1|Sex=female) = 0.290363\)

  • \(f(Pclass=2|Sex=female) = 0.242038\)

  • \(f(Pclass=3|Sex=female) = 0.458599\)

These are the frequencies of women in the three classes. Similarly, the second row represents the conditional frequencies with respect to Sex=male. From the table, we notice that the distribution of passengers changes across the three classes. In particular, among men, the third class is more frequent than among women.

We can obtain the complementary perspective by conditioning on class instead:

# normalize=1 indicates conditioning on the first variable
pd.crosstab(titanic['Sex'], titanic['Pclass'], normalize=1)
Pclass 1 2 3
Sex
female 0.435185 0.413043 0.293279
male 0.564815 0.586957 0.706721

In this case, each column will be a probability distribution. For example:

  • \(P(Sex=female|Pclass=1) = 0.435185\)

  • \(P(Sex=male|Pclass=1) = 0.564815\)

We note that the proportion between men and women changes in the three classes, and in particular in the third class there are many more men than women.

Graphical Representations#

We can easily obtain graphical representations of the relationships between two variables using bar charts, directly from contingency tables.

We can compare the absolute frequencies of passengers and sexes in the three classes as follows:

pd.crosstab(titanic['Sex'], titanic['Pclass']).plot.bar()

<AxesSubplot:xlabel='Sex'>

From this, we note that many passengers are men and in third class. In these cases, a stacked plot can sometimes be useful:

pd.crosstab(titanic['Sex'], titanic['Pclass']).plot.bar(stacked=True)

<AxesSubplot:xlabel='Sex'>

It is often useful to visualize conditional distributions. For example, the following graph allows us to compare the distributions of passengers in the three classes, dividing them into two groups based on sex:

pd.crosstab(titanic['Sex'], titanic['Pclass'], normalize=0).plot.bar()

<AxesSubplot:xlabel='Sex'>

We can obtain the graph of conditional frequencies with respect to the classes for a complementary view. In this case, to have a meaningful graph, we need to transpose the contingency table with a .T:

# normalize=1 indicates conditioning on the first variable
pd.crosstab(titanic['Sex'], titanic['Pclass'], normalize=1).T.plot.bar()

<AxesSubplot:xlabel='Pclass'>

Measures of Association Between Two Discrete Variables#

Pearson’s \(\mathcal{X}^2\) Statistic#

We can calculate Pearson’s \(\mathcal{X}^2\) statistic as follows:

from scipy.stats import chi2_contingency
contingency = pd.crosstab(titanic['Pclass'], titanic['Survived'])
chi2_contingency(contingency)

Chi2ContingencyResult(statistic=102.88898875696056, pvalue=4.549251711298793e-23, dof=2, expected_freq=array([[133.09090909,  82.90909091],
       [113.37373737,  70.62626263],
       [302.53535354, 188.46464646]]))

The chi2_contingency function returns:

  • The statistic value;

  • the p-value (we will see this in more detail later);

  • the degrees of freedom of the \(\chi^2\) distribution (we will also see this in more detail later);

  • The expected frequencies under the assumption of independence.

We can compare these frequencies with the observed ones:

```python
def calculate_contingency_table(group1_data, group2_data):
    """
    Calcola la tabella di contingenza per due gruppi di dati.

    Args:
        group1_data (list): Dati per il primo gruppo.
        group2_data (list): Dati per il secondo gruppo.

    Returns:
        dict: Tabella di contingenza.
    """
    # Inizializza la tabella di contingenza
    contingency_table = {
        "group1_present_group2_present": 0,
        "group1_present_group2_absent": 0,
        "group1_absent_group2_present": 0,
        "group1_absent_group2_absent": 0
    }

    # Itera sui dati per popolare la tabella
    for i in range(len(group1_data)):
        if group1_data[i] and group2_data[i]:
            contingency_table["group1_present_group2_present"] += 1
        elif group1_data[i] and not group2_data[i]:
            contingency_table["group1_present_group2_absent"] += 1
        elif not group1_data[i] and group2_data[i]:
            contingency_table["group1_absent_group2_present"] += 1
        else:
            contingency_table["group1_absent_group2_absent"] += 1

    return contingency_table

def display_contingency_table(table):
    """
    Visualizza la tabella di contingenza in un formato leggibile.

    Args:
        table (dict): Tabella di contingenza da visualizzare.
    """
    print("Tabella di Contingenza:")
    print(f"  Presente (Gruppo 1) | Assente (Gruppo 1)")
    print("---------------------------------------------")
    print(f"Presente (Gruppo 2): {table['group1_present_group2_present']:<10} | Assente (Gruppo 2): {table['group1_present_group2_absent']:<10}")
    print(f"Assente (Gruppo 2):  {table['group1_absent_group2_present']:<10} | Assente (Gruppo 2): {table['group1_absent_group2_absent']:<10}")

# Esempio di utilizzo
dati_gruppo_a = [True, False, True, True, False, True, False, False]
dati_gruppo_b = [True, True, False, True, False, False, True, False]

tabella_risultante = calculate_contingency_table(dati_gruppo_a, dati_gruppo_b)
display_contingency_table(tabella_risultante)
```

Cramer’s V Statistic#

Cramer’s V statistic can be calculated as follows:

from scipy.stats.contingency import association

association(pd.crosstab(titanic['class'], titanic['survived']))

0.33981738800531175

Relative Risk and Odds Ratio#

Let’s consider this contingency table:

import pandas as pd
contingency = pd.DataFrame({
    "Exposed": [20, 380],
    "Non Exposed": [6,594]
}, index=["Diseased", "Healthy"])
contingency
Exposed Non Exposed
Diseased 20 6
Healthy 380 594

We can calculate the relative risk as follows:

from scipy.stats.contingency import relative_risk
relative_risk(contingency['Exposed']['Diseased'],contingency['Exposed'].sum(),contingency['Non Exposed']['Diseased'], contingency['Non Exposed'].sum())

RelativeRiskResult(relative_risk=5.0, exposed_cases=20, exposed_total=400, control_cases=6, control_total=600)

The function requires that the various necessary values be passed rather than the contingency matrix to avoid ambiguity.

The odds ratio is calculated as follows:

from scipy.stats.contingency import odds_ratio
odds_ratio(contingency)

OddsRatioResult(statistic=5.202179487843056)

Scatter Plot#

We can show a scatterplot in seaborn as follows:

from matplotlib import pyplot as plt
plt.figure(figsize=(8,6))
sns.scatterplot(x=iris['sepal_width'], y=iris['sepal_length'])
plt.show()

Alternatively, we can use Pandas:

iris.plot.scatter(x='sepal_width',y='sepal_length')
plt.show()

With seabord, we can easily color the points differently depending on their belonging to different iris classes using the hue parameter:

plt.figure(figsize=(8,6))
sns.scatterplot(x=iris['sepal_width'], y=iris['sepal_length'], hue=iris['species'])
plt.show()

plt.figure(figsize=(8,6))

As we can notice from the two plots, each pair of variables allows us to see “a part” of the possible interactions between the different variables. sns.pairplot(iris)

Means and Covariances#

In the multivariate case, the mean is calculated in a manner entirely analogous to the univariate case as the mean of vectors. We can trivially calculate the mean of our multivariate data using the mean method:

print(iris.mean())

sepal_length    5.843333
sepal_width     3.057333
petal_length    3.758000
petal_width     1.199333
dtype: float64

/var/folders/cs/p62_d78d49n3ddj0xlfh1h7r0000gn/T/ipykernel_34355/769219605.py:1: FutureWarning: Dropping of nuisance columns in DataFrame reductions (with 'numeric_only=None') is deprecated; in a future version this will raise TypeError.  Select only valid columns before calling the reduction.
  print(iris.mean())

The result will be a vector (a Pandas Series) that identifies a point in four-dimensional space representing the average of the observations. Let’s see a simple two-dimensional example, considering only a few variables:

iris2d = iris[['sepal_length','sepal_width']]
mean_point = iris2d.mean()
plt.scatter(iris2d['sepal_length'],iris2d['sepal_width'])
plt.plot(mean_point.iloc[0], mean_point.iloc[1],'rx')
plt.show()

The red cross represents the midpoint of the two-dimensional dataset considered.
