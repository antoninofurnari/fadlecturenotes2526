Absolute and Relative Frequencies#

We can obtain the unique values in a variable with the unique() function:

data['sex'].unique()

array(['M', 'F'], dtype=object)

Absolute frequencies are calculated using Pandas’ value_counts method. For example, let’s visualize the absolute frequencies of the sex values:

sex_counts = data['sex'].value_counts()

A good way to understand if it makes sense to calculate the absolute frequencies for a given variable is to check how many unique values appear. This can be done using the unique() function:

Bar Chart of Absolute Frequencies#

Data frequencies can be graphically represented using a bar chart by utilizing the plotting functionalities of Pandas. These functionalities rely on matplotlib, therefore, to have greater control over them we must import pyplot:

from matplotlib import pyplot as plt
data['height'].value_counts().sort_index().plot.bar(figsize=(18,6))
plt.grid()
plt.show()

where the parameter figsize=(12,6) serves to specify the dimensions of the plot, while plt.grid() serves to display a grid superimposed on the plot.

The resulting graph is called a bar chart of absolute frequencies and shows the number of times each value appears in the sample.

Question 2

What does the data representation above tell us? What are the most frequent values? How many people have a height of \(167.64\ cm\)?

Note that, alternatively, it is possible to plot a bar graph directly via matplotlib as follows:

plt.figure(figsize=(18,6))
height_counts = data['height'].value_counts().sort_index()
plt.bar(height_counts.index, height_counts.values)
plt.xticks(height_counts.index, rotation='vertical')
plt.grid()
plt.show()

In many cases it will be possible to use the compact functions provided by Pandas, while in others it will be necessary to use matplotlib explicitly.

Relative frequency bar charts can also be useful for comparing different data samples. For example, let’s consider the heights of women and men as two separate samples in the dataset and calculate their relative frequencies:

height_pmf_m = data[data['sex']=='M']['height'].value_counts(normalize=True).sort_index()
height_pmf_f = data[data['sex']=='F']['height'].value_counts(normalize=True).sort_index()

We show the two histograms by comparing them. In order to easily display the two histograms in the same graph, we will use matplotlib:

plt.figure(figsize=(18,6))

plt.bar(pmf_height_m.index+0.2, pmf_height_m.values, width=1, alpha=0.9)

Empirical Cumulative Distribution Function (ECDF)#

We can calculate an ECDF using the cumsum function, which, given a series of values, calculates the sum of the first \(i\) values at each position \(i\). For example:

a = pd.Series([1,3,6,8,9])
a.cumsum()

0     1
1     4
2    10
3    18
4    27
dtype: int64

Since the relative frequencies are ordered, the cumsum function calculates for each index the sum of the relative frequencies of the previous values. We calculate the ECDF of the two samples and compare them:

ecdf_weight_m = data[data['sex']=='M']['weight'].value_counts(normalize=True).sort_index().cumsum()
ecdf_weight_f = data[data['sex']=='F']['weight'].value_counts(normalize=True).sort_index().cumsum()

plt.figure(figsize=(12,8))
plt.plot(ecdf_weight_m.index, ecdf_weight_m.values)
plt.plot(ecdf_weight_f.index, ecdf_weight_f.values)
plt.legend(['M','F'])
plt.grid()
plt.show()

Histograms#

We can plot a histogram with matplotlib as follows:

plt.figure(figsize=(12,6))
_,edges,_=plt.hist(data['weight'], width=10.8)
plt.xticks(edges)
plt.grid()
plt.show()

In addition to plotting the histogram, the plt.hist function returns three values:

  • a vector containing the histogram values;

  • the “edges” of each bin;

  • a list of handles to the displayed graphical elements.

In the example above, we used the notation _,edges,_=... to discard the first and last values and obtain the histogram “edges”.

Sturges and Rice#

Let’s see how to represent histograms by choosing bin numbers with Sturges and Rice:

import numpy as np
bins_struges=int(3.3*np.log(len(data['weight'])))
bins_rice=int(2*len(data['weight'])**(1/3))

plt.figure(figsize=(12,4))
plt.subplot(1,2,1)
plt.title('Struges ({} bins)'.format(bins_struges))
plt.hist(data['weight'], bins=bins_struges)
plt.grid()
plt.subplot(1,2,2)
plt.title('Rice ({} bins)'.format(bins_rice))
plt.grid()
plt.hist(data['weight'], bins=bins_rice)
plt.show()

It’s also possible to explicitly specify the “edges” of the bins:

plt.figure(figsize=(12,6))
_,edges,_=plt.hist(data['weight'], bins=[40,50,60,70,80,90,100,110,120,130],
                   width=9.5) # build a histogram with defined bins
plt.xticks(edges)
plt.grid()
plt.show()

The histogram shown above reports the absolute frequencies for each bin and allows us to answer questions like “how many subjects have a weight between \(70 Kg\) and \(80 Kg\)”?#

Density Histograms#

To obtain a density histogram, we must pass the density=True parameter to the hist function:

plt.figure(figsize=(12,6))
values,edges,_=plt.hist(data['weight'], bins=[40,50,60,70,80,90,100,110,120,130], 
                   width=9.5, density=True)
plt.xticks(edges)
plt.grid()
plt.show()

We can verify that the condition

\[ \sum_{i=0}^{n} b_i \cdot w_i = 1 \]

is indeed satisfied:

widths = edges[1:] - edges[:-1]
print(widths)  # all bins have the same width
print((widths * values).sum())  # calculate the sum of products

[10 10 10 10 10 10 10 10 10]
1.0

Question 7

Use the histogram above to obtain the probability that a subject has a weight between \(40Kg\) and \(50Kg\).

Histograms can also be plotted directly from DataFrame:

data['weight'].plot.hist(figsize=(12,6), width=9.5, 
                         density='True',
                         bins=[40,50,60,70,80,90,100,110,120])
plt.grid()
plt.show()

The function takes the same input parameters as matplotlib’s hist function. However, unlike it, it does not return the bin values and their respective edges.

Density Estimation#

We can estimate and plot densities using Pandas as follows:

data['weight'].plot.density(figsize=(12,6))
plt.grid()
plt.show()

Comparing Samples Using Histograms#

Histograms can be useful for comparing samples. Let’s compare the histograms of the sample of men’s weights and the sample of women’s weights:

plt.figure(figsize=(12,6))
plt.hist(data[data['sex']=='F']['weight'], width=7, alpha=0.9)
plt.hist(data[data['sex']=='M']['weight'], width=7, alpha=0.9)
plt.legend(['F','M'])
plt.grid()
plt.show()

Barplot#

In addition to showing data frequencies, bar plots can be useful for comparing different numerical quantities. Suppose we want to compare the number of women and men on board the Titanic. We can calculate these numbers using a groupby:

titanic.groupby('Sex')['Sex'].count()

We can compare these two quantities visually using a bar plot as follows:

titanic.groupby('Sex')['Sex'].count().plot.bar()
plt.ylabel('Number of Passengers')
plt.show()

Let’s now compare the average ages of passengers boarded in the various classes. First, let’s build a summary table using groupby:

titanic.groupby('Pclass')['Age'].mean()

We can compare the quantities obtained graphically using a barplot:

titanic.groupby('Pclass')['Age'].mean().plot.bar()
plt.ylabel('Age')

We calculate the number of survivors by boarding class:

pd.crosstab(titanic['Pclass'], titanic['Survived'])

Pclass

To obtain percentages, we can normalize by rows as follows:

pd.crosstab(titanic['Class'], titanic['Survived'], normalize='index')
Survived 0 1
Pclass
1 0.370370 0.629630
2 0.527174 0.472826

Each row of the table shows the percentage of survivors (1) and the percentage of non-survivors (0). Let’s visualize a bar graph of the table:

pd.crosstab(titanic['pclass'], titanic['survived'], normalize='index').plot.bar()
plt.ylabel('Survived %')
plt.show()

A plot often useful for comparing complementary quantities (as in the case of percentages seen above) is the “stacked bar plot”. To obtain this plot, you need to pass stacked=True to the bar method:

pd.crosstab(titanic['Pclass'], titanic['Survived'], normalize='index').plot.bar(stacked=True)
plt.ylabel('%')
plt.show()

We can also relate more variables. For example:
